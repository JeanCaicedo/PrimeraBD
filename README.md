# PrimeraBD

![SQLite](https://img.shields.io/badge/SQLite-07405E?style=for-the-badge&logo=sqlite&logoColor=white)
![SQL](https://img.shields.io/badge/SQL-4479A1?style=for-the-badge&logo=postgresql&logoColor=white)

Primera base de datos creada desde cero con SQLiteStudio. Es literalmente el "hola mundo" de bases de datos: una tabla de usuarios para practicar la creación de esquemas y registros en SQLite.

Es un ejercicio muy básico de aprendizaje, hecho cuando apenas empezaba a familiarizarme con el manejo de bases de datos relacionales.

## Stack

- SQLite como motor de base de datos
- SQLiteStudio como herramienta de administración

## Estructura

```
PrimeraBD/
├── usuarios.db        # Archivo binario de la base de datos SQLite
├── usuarios.sqbpro    # Proyecto de SQLiteStudio
└── README.md
```

## Cómo abrirlo

1. Instalar [SQLiteStudio](https://sqlitestudio.pl/).
2. Abrir el archivo `usuarios.sqbpro` desde SQLiteStudio (o importar `usuarios.db` directamente como base SQLite).
3. Explorar la tabla `usuarios`.

## Notas

- Ejercicio de aprendizaje, no está pensado para producción.
- Se conserva como memoria del primer contacto con bases de datos.
